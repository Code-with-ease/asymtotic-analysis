# asymtotic-analysis
Fuzzy trafic manager using graphviz.

## How to install on your ubuntu system
```
sudo apt-get install graphviz

git clone https://github.com/Code-with-ease/road-not-taken.git

g++ setup.cpp -std=c++11 -a.o

./a.o
```
